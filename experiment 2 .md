#  Basic Linux File Commands: `cp`, `mv`, and `touch`

This guide explains three essential file management commands in Linux:

- `touch` – create an empty file
- `cp` – copy files or directories
- `mv` – move (or rename) files or directories

---

##  1. `touch' Command

###  Purpose:
Creates an empty file.

###  Syntax:
```bash
touch filename
touch hello.txt
```
## 2. 'cp' command

### purpose:
the cp command is used to copy files or directories.
### syntax:
```bash
cp source_file target_file
cp hello.txt copy_hello.txt
```

## 3. 'mv' command

### purpose:
mv command is used to move files.

### syntax:
```bash
mv source_file target_location
mv hello.txt Documents/
```
## 4. 'touch ' command

### purpose:
creates multiple file
### syntax:
```bash
touch file1.txt
```
## 5. 'mkdir' command

### purpose:
make new directories.

### syntax :
```bash
mkdir target_folder
```
## 6. 'ls' command

### purpose:
creates a target folder.

### syntax:
```bash
ls target_folder/
file1.txt
```
## 7. 'cp' command

### purpose:
copies files and folders.

### syntax:
```bash
cp -r folder1 folder2
mv -r folder1 folder2
```
